Zuhause am Bach - Gaeste-App V49 Smart-WLAN

Installation:
1. ZIP entpacken.
2. Alle Dateien im entpackten Ordner zusammen lassen.
3. index.html im Browser oeffnen oder den kompletten Ordner auf den Webserver/GitHub Pages hochladen.

Enthalten:
- index.html
- app.js
- style.css
- hero-windis.png

Hinweise:
- Wetter und Diagramm brauchen Internetzugang.
- Die Wetterdaten kommen von Open-Meteo.
- Das Diagramm nutzt Chart.js ueber CDN.
- Die App ist eine statische Webseite und braucht keine Server-Installation.

Bereinigt in dieser Install-ZIP:
- Veraltete Booking-Daten entfernt.
- Platzhalter-Sprachwahl entfernt.
- Sichtbaren Entwickler-App-Check entfernt.
- Alte Test-, Report- und Logdateien entfernt.
